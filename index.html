Turing Pi RK1 - Armbian Kernel APT Repository

Automatically updated APT repository for Armbian kernel packages targeting the Turing Pi RK1 (Rockchip RK3588S). The two most recent kernel versions are always available.

Add this repository

echo "deb [trusted=yes] https://Jean-Pereira-Dev.github.io/turing-pi_rk1 bookworm main" | sudo tee /etc/apt/sources.list.d/turing-rk1-kernel.list
sudo apt update

Install the kernel

sudo apt install linux-image-current-rockchip64 linux-dtb-current-rockchip64 linux-headers-current-rockchip64
sudo reboot
